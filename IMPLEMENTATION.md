# Full Runtime SDUI Implementation Specification

## tauri-shared Rust Crate

---

## 1. Overview

### Purpose in SDUI Stack

`tauri-shared` provides the shared Rust type definitions, runtime engine, and Tauri commands that underpin Full Runtime SDUI across all 8 Tauri mobile/desktop applications. It serves as the single source of truth for schema types, the SDUI rendering engine, and the bridge between persisted schema data and the Angular frontend.

### Connection to tauri-front-shared (Angular)

The Angular frontend (`tauri-front-shared`) consumes the TypeScript bindings generated by `ts-rs` from these Rust types. The generated bindings live in `tauri-shared/bindings/` and are imported as an npm package in the Angular workspace. The runtime engine in Rust executes SDUI commands invoked from Angular via Tauri IPC:

```
Angular (tauri-front-shared)
    |
    | tauri::invoke("sdui_load_schema", ...)
    v
Tauri Commands (tauri-shared/src/commands/sdui_commands.rs)
    |
    | uses
    v
SduiEngine (tauri-shared/src/runtime/engine.rs)
    |
    | reads/writes
    v
SignalStore + JsonDb (tauri-shared/src/storage/)
```

### What's Missing for Runtime SDUI

1. **Runtime engine** (`src/runtime/`) — no `SduiEngine`, `SchemaRouter`, `BindingResolver`, or validation logic
2. **SDUI Tauri commands** — no `sdui_*` commands for schema loading, binding resolution, event dispatching, page rendering, or theme application
3. **Runtime schema types** — no `RenderedPage`, `ValidationResult`, `ValidationError`, `EventSignature`
4. **Module registration** — `src/lib.rs` does not export the new `runtime` or `sdui_commands` modules
5. **TypeScript bindings for runtime types** — runtime structs need `#[derive(TS)]` for Angular consumption

### 3b. `src/runtime/engine.rs`

```rust
use std::collections::HashMap;
use std::sync::Arc;

use serde_json::Value;
use schemars::JsonSchema;
use serde::{Deserialize, Serialize};
use ts_rs::TS;

use crate::schema::{UiSchema, Theme, Page, Layout, ComponentDef};
use crate::storage::SignalStore;
use crate::AppError;
use crate::Result;

/// A callable function registered in the engine.
pub type Callable = Arc<dyn Fn(Vec<Value>) -> Result<Value> + Send + Sync>;

/// A route guard function: returns `Ok(true)` to allow navigation.
pub type GuardFn = Arc<dyn Fn(&str) -> Result<bool> + Send + Sync>;

/// A middleware function applied before page render.
pub type MiddlewareFn = Arc<dyn Fn(&mut RenderedPage) -> Result<()> + Send + Sync>;

/// The central SDUI runtime engine.
///
/// Owns the loaded [`UiSchema`], active [`Theme`], [`SignalStore`] for reactive
/// state, and a registry of callable JS/TS functions exposed to the schema.
#[derive(Debug, Clone)]
pub struct SduiEngine {
    /// The fully-loaded schema for the current app session.
    pub schema: UiSchema,
    /// The currently active theme.
    pub theme: Theme,
    /// Reactive state store. Pages and components subscribe to keys here.
    pub store: Arc<SignalStore>,
    /// Registry of callable functions available to data bindings and expressions.
    pub functions: HashMap<String, Callable>,
}

impl SduiEngine {
    /// Creates a new engine with the given schema, theme, and empty store/functions.
    pub fn new(schema: UiSchema, theme: Theme) -> Self {
        Self {
            schema,
            theme,
            store: Arc::new(SignalStore::new()),
            functions: HashMap::new(),
        }
    }

    /// Creates a new engine with an existing [`SignalStore`] (e.g. pre-populated from disk).
    pub fn with_store(schema: UiSchema, theme: Theme, store: Arc<SignalStore>) -> Self {
        Self {
            schema,
            theme,
            store,
            functions: HashMap::new(),
        }
    }

    /// Loads a new schema into the engine, replacing the current one.
    pub fn load_schema(&mut self, schema: UiSchema) {
        self.schema = schema;
    }

    /// Returns a reference to the current schema.
    pub fn schema(&&self) -> &UiSchema {
        &self.schema
    }

    /// Returns a reference to the current theme.
    pub fn theme(&self) -> &Theme {
        &self.theme
    }

    /// Applies a new theme, replacing `self.theme`.
    pub fn apply_theme(&mut self, theme: Theme) {
        self.theme = theme;
    }

    /// Returns the [`Page`] with the given `page_id`, or `None` if not found.
    pub fn get_page(&self, page_id: &str) -> Option<&Page> {
        self.schema.pages.iter().find(|p| p.id == page_id)
    }

    /// Returns the [`Layout`] with the given `layout_id`, or `None` if not found.
    pub fn get_layout(&self, layout_id: &str) -> Option<&Layout> {
        self.schema.layouts.iter().find(|l| l.id == layout_id)
    }

    /// Returns the [`ComponentDef`] with the given `component_id`, or `None`.
    pub fn get_component(&self, component_id: &str) -> Option<&ComponentDef> {
        self.schema
            .components
            .iter()
            .chain(self.schema.shared_components.iter())
            .find(|c| c.id == component_id)
    }

    /// Registers a callable function under `name`.
    pub fn register_function<F>(&mut self, name: String, callable: F)
    where
        F: Fn(Vec<Value>) -> Result<Value> + Send + Sync + 'static,
    {
        self.functions.insert(name, Arc::new(callable));
    }

    /// Calls a registered function by `name` with the given arguments.
    pub fn call_function(&self, name: &str, args: Vec<Value>) -> Result<Value> {
        self.functions
            .get(name)
            .ok_or_else(|| AppError::NotFound(format!("Function '{}' not found", name)))?(args)
    }

    /// Returns `true` if a function with `name` is registered.
    pub fn has_function(&self, name: &str) -> bool {
        self.functions.contains_key(name)
    }

    /// Returns the current value for a store `key`.
    pub fn get_state(&self, key: &str) -> Option<Value> {
        self.store.get(key)
    }

    /// Sets a store `key` to `value`, notifying all subscribers.
    pub fn set_state(&self, key: &str, value: Value) {
        self.store.set(key, value);
    }

    /// Updates a store `key` using the given `f` function.
    pub fn update_state<F>(&self, key: &str, f: F)
    where
        F: FnOnce(&Value) -> Value,
    {
        self.store.update(key, f);
    }

    /// Deletes a store `key`.
    pub fn delete_state(&self, key: &str) {
        self.store.delete(key);
    }

    /// Subscribes to store changes. The callback receives `(key, value)` on each mutation.
    pub fn subscribe_to_state<F>(&self, callback: F)
    where
        F: Fn(&str, &Value) + Send + Sync + 'static,
    {
        self.store.subscribe(callback);
    }

    /// Serializes the entire store to a JSON `Value`.
    pub fn export_state(&self) -> Value {
        self.store.to_json()
    }

    /// Imports a JSON `Value` into the store, replacing all existing keys.
    pub fn import_state(&self, json: Value) {
        self.store.from_json(json);
    }

    /// Returns the list of all store keys.
    pub fn state_keys(&self) -> Vec<String> {
        self.store.keys()
    }
}

impl From<UiSchema> for SduiEngine {
    fn from(schema: UiSchema) -> Self {
        let theme = schema
            .app
            .settings
            .themes
            .first()
            .cloned()
            .unwrap_or_else(|| crate::schema::theme::get_light_theme());
        Self::new(schema, theme)
    }
}
```

### 3c. `src/runtime/router.rs`

```rust
use std::collections::HashMap;
use std::sync::Arc;

use schemars::JsonSchema;
use serde::{Deserialize, Serialize};
use ts_rs::TS;

use crate::runtime::engine::{GuardFn, MiddlewareFn, SduiEngine};
use crate::runtime::schema_types::RenderedPage;
use crate::Result;

use super::engine::Callable;

/// Configuration for a single route resolved by [`SchemaRouter`].
#[derive(Debug, Clone, Serialize, Deserialize, JsonSchema, TS)]
#[serde(rename_all = "camelCase")]
#[ts(export)]
pub struct RouteConfig {
    /// The page to render when this route matches.
    pub page_id: String,
    /// Optional layout wrapper (layout ID) to apply.
    #[serde(default)]
    pub layout_id: Option<String>,
    /// Guard functions evaluated before navigation is allowed.
    #[serde(default)]
    pub guards: Vec<String>,
    /// Middleware functions evaluated after guards, before render.
    #[serde(default)]
    pub middleware: Vec<String>,
}

/// Routes incoming route paths to [`RouteConfig`] and resolves them to
/// a fully rendered [`RenderedPage`] using the shared [`SduiEngine`].
#[derive(Debug, Clone)]
pub struct SchemaRouter {
    routes: HashMap<String, RouteConfig>,
    engine: Arc<SduiEngine>,
}

impl SchemaRouter {
    /// Creates a new router backed by the given engine and an empty route table.
    pub fn new(engine: Arc<SduiEngine>) -> Self {
        Self {
            routes: HashMap::new(),
            engine,
        }
    }

    /// Creates a router with an pre-populated route map.
    pub fn with_routes(engine: Arc<SduiEngine>, routes: HashMap<String, RouteConfig>) -> Self {
        Self { routes, engine }
    }

    /// Registers a `path` -> `config` mapping.
    pub fn add_route(&mut self, path: String, config: RouteConfig) {
        self.routes.insert(path, config);
    }

    /// Removes the route registered at `path`.
    pub fn remove_route(&mut self, path: &str) {
        self.routes.remove(path);
    }

    /// Returns a reference to the route config for `path`, if any.
    pub fn get_route(&self, path: &str) -> Option<&RouteConfig> {
        self.routes.get(path)
    }

    /// Returns an iterator over all registered routes.
    pub fn routes(&self) -> impl Iterator<Item = (&String, &RouteConfig)> {
        self.routes.iter()
    }

    /// Resolves `path` to a [`RenderedPage`].
    ///
    /// Steps:
    /// 1. Look up `RouteConfig` for `path`.
    /// 2. Evaluate all guard functions; return error if any returns `false`.
    /// 3. Look up the `Page` from the engine's schema.
    /// 4. Apply the page's layout (if `layout_id` is set in config).
    /// 5. Resolve all data bindings for all canvas elements.
    /// 6. Apply middleware functions in order.
    /// 7. Return the rendered page.
    pub fn resolve(&self, path: &str) -> Result<RenderedPage> {
        let config = self
            .routes
            .get(path)
            .ok_or_else(|| crate::AppError::NotFound(format!("Route '{}' not found", path)))?;

        for guard_name in &config.guards {
            let guard = self
                .engine
                .functions
                .get(guard_name)
                .ok_or_else(|| crate::AppError::NotFound(format!("Guard '{}'", guard_name)))?;
            let result = guard(vec![])?;
            if !result.as_bool().unwrap_or(false) {
                return Err(crate::AppError::Forbidden);
            }
        }

        let page = self
            .engine
            .get_page(&config.page_id)
            .ok_or_else(|| crate::AppError::NotFound(format!("Page '{}'", config.page_id)))?;

        let mut rendered = RenderedPage::from_page(page, config.layout_id.as_deref());

        let binding_resolver = super::bindings::BindingResolver::new(
            self.engine.store.clone(),
            self.engine.functions.clone(),
        );
        rendered.resolve_bindings(&binding_resolver)?;

        for mw_name in &config.middleware {
            let mw = self
                .engine
                .functions
                .get(mw_name)
                .ok_or_else(|| crate::AppError::NotFound(format!("Middleware '{}'", mw_name)))?;
            mw(serde_json::value::to_value(&mut rendered).unwrap_or_default())?;
        }

        Ok(rendered)
    }

    /// Returns `true` if a route is registered at `path`.
    pub fn has_route(&self, path: &str) -> bool {
        self.routes.contains_key(path)
    }

    /// Returns the number of registered routes.
    pub fn route_count(&self) -> usize {
        self.routes.len()
    }

    /// Returns a reference to the backing engine.
    pub fn engine(&self) -> &Arc<SduiEngine> {
        &self.engine
    }
}
```

### 3d. `src/runtime/bindings.rs`

```rust
use std::collections::HashMap;
use std::sync::Arc;

use serde_json::Value;
use schemars::JsonSchema;
use serde::{Deserialize, Serialize};

use crate::runtime::engine::Callable;
use crate::runtime::schema_types::RenderedPage;
use crate::schema::{CanvasElement, DataBinding};
use crate::storage::SignalStore;
use crate::Result;

/// Resolves [`DataBinding`] expressions against the reactive [`SignalStore`]
/// and the callable function registry.
#[derive(Debug, Clone)]
pub struct BindingResolver {
    store: Arc<SignalStore>,
    functions: Arc<HashMap<String, Callable>>,
}

impl BindingResolver {
    /// Creates a new resolver with the given store and function registry.
    pub fn new(store: Arc<SignalStore>, functions: Arc<HashMap<String, Callable>>) -> Self {
        Self { store, functions }
    }

    /// Resolves a single [`DataBinding`] expression.
    ///
    /// The `entity` field names a store key. If `field` is `Some(f)`,
    /// the resolver navigates to that sub-key within the stored object.
    pub fn resolve_binding(&self, binding: &DataBinding) -> Result<Option<Value>> {
        let entity_data = self
            .store
            .get(&binding.entity)
            .ok_or_else(|| crate::AppError::NotFound(format!("Entity '{}' not found in store", binding.entity)))?;

        if let Some(field) = &binding.field {
            entity_data
                .get(field)
                .cloned()
                .Ok_or_else(|| crate::AppError::NotFound(format!("Field '{}' on entity '{}'", field, binding.entity)))
                .map(Some)
        } else {
            Ok(Some(entity_data))
        }
    }

    /// Resolves a binding by entity name only (no field navigation).
    pub fn resolve_entity(&self, entity_name: &str) -> Result<Option<Value>> {
        Ok(self.store.get(entity_name))
    }

    /// Resolves all bindings for a single [`CanvasElement`].
    ///
    /// If the element has no `data_binding`, returns `Ok(None)`.
    pub fn resolve_element_binding(&self, element: &CanvasElement) -> Result<Option<Value>> {
        match &element.data_binding {
            Some(binding) => self.resolve_binding(binding),
            None => Ok(None),
        }
    }

    /// Resolves a dot-notation path (e.g. `"user.address.city"`) against the store.
    pub fn resolve_path(&self, path: &str) -> Result<Option<Value>> {
        let parts: Vec<&str> = path.split('.').collect();
        let mut current = self
            .store
            .get(parts[0])
            .ok_or_else(|| crate::AppError::NotFound(format!("Path root '{}' not found", parts[0])))?;

        for part in parts.iter().skip(1) {
            current = current
                .get(*part)
                .cloned()
                .ok_or_else(|| crate::AppError::NotFound(format!("Path segment '{}' in '{}'", part, path)))?;
        }
        Ok(Some(current))
    }

    /// Calls a registered function by `name` with `args`.
    pub fn call_function(&self, name: &str, args: Vec<Value>) -> Result<Value> {
        self.functions
            .get(name)
            .ok_or_else(|| crate::AppError::NotFound(format!("Function '{}' not found", name)))?(args)
    }

    /// Returns `true` if the store contains `key`.
    pub fn has_entity(&self, key: &str) -> bool {
        self.store.get(key).is_some()
    }

    /// Returns the set of all entity keys currently in the store.
    pub fn entity_keys(&self) -> Vec<String> {
        self.store.keys()
    }

    /// Returns `true` if a function with `name` is registered.
    pub fn has_function(&self, name: &str) -> bool {
        self.functions.contains_key(name)
    }
}

/// Extension trait for [`RenderedPage`] to resolve all bindings on a rendered page.
pub trait ResolveBindings {
    fn resolve_bindings(&mut self, resolver: &BindingResolver) -> Result<()>;
}

impl ResolveBindings for RenderedPage {
    fn resolve_bindings(&mut self, resolver: &BindingResolver) -> Result<()> {
        for element in &mut self.canvas_elements {
            if let Some(binding) = &element.data_binding {
                element.resolved_props = resolver.resolve_binding(binding)?;
            }
            for child_id in &element.children {
                // child resolution is handled recursively via the page structure
                let _ = child_id;
            }
        }
        Ok(())
    }
}
```

### 3e. `src/runtime/validation.rs`

```rust
use schemars::JsonSchema;
use serde::{Deserialize, Serialize};
use ts_rs::TS;

use crate::schema::{
    UiSchema, Page, ComponentDef, CanvasElement, Layout, ServiceDef, ModuleDef,
};
use crate::runtime::schema_types::{ValidationResult, ValidationError};

/// Validates a complete [`UiSchema`].
///
/// Checks:
/// - All referenced pages exist
/// - All referenced layouts exist
/// - All referenced components exist
/// - All service entity references are valid
/// - All module command references are valid
/// - No duplicate IDs across pages, layouts, components
pub fn validate_schema(schema: &UiSchema) -> ValidationResult {
    let mut errors: Vec<ValidationError> = Vec::new();

    let mut seen_ids: std::collections::HashSet<&str> = std::collections::HashSet::new();

    for page in &schema.pages {
        if let Err(errs) = validate_page(page, schema) {
            errors.extend(errs);
        }
        if !seen_ids.insert(page.id.as_str()) {
            errors.push(ValidationError {
                path: format!("pages[{}]", page.id),
                message: format!("Duplicate page ID: '{}'", page.id),
                error_type: "duplicate".to_string(),
            });
        }
    }

    for layout in &schema.layouts {
        if !seen_ids.insert(layout.id.as_str()) {
            errors.push(ValidationError {
                path: format!("layouts[{}]", layout.id),
                message: format!("Duplicate layout ID: '{}'", layout.id),
                error_type: "duplicate".to_string(),
            });
        }
    }

    for component in schema.components.iter().chain(schema.shared_components.iter()) {
        if !seen_ids.insert(component.id.as_str()) {
            errors.push(ValidationError {
                path: format!("components[{}]", component.id),
                message: format!("Duplicate component ID: '{}'", component.id),
                error_type: "duplicate".to_string(),
            });
        }
    }

    for service in &schema.services {
        if !seen_ids.insert(service.id.as_str()) {
            errors.push(ValidationError {
                path: format!("services[{}]", service.id),
                message: format!("Duplicate service ID: '{}'", service.id),
                error_type: "duplicate".to_string(),
            });
        }
    }

    for module in &schema.modules {
        if !seen_ids.insert(module.id.as_str()) {
            errors.push(ValidationError {
                path: format!("modules[{}]", module.id),
                message: format!("Duplicate module ID: '{}'", module.id),
                error_type: "duplicate".to_string(),
            });
        }
    }

    ValidationResult {
        valid: errors.is_empty(),
        errors,
    }
}

/// Validates a single [`Page`]: its layout reference and all canvas elements.
pub fn validate_page(page: &Page, schema: &UiSchema) -> Result<Vec<ValidationError>, Vec<ValidationError>> {
    let mut errors: Vec<ValidationError> = Vec::new();

    if let Some(layout_id) = &page.layout {
        if !schema.layouts.iter().any(|l| &l.id == layout_id) {
            errors.push(ValidationError {
                path: format!("pages.{}.layout", page.id),
                message: format!("Referenced layout '{}' not found", layout_id),
                error_type: "reference".to_string(),
            });
        }
    }

    for element in &page.canvas_elements {
        if let Err(errs) = validate_canvas_element(element, schema) {
            errors.extend(errs);
        }
    }

    if errors.is_empty() {
        Ok(vec![])
    } else {
        Err(errors)
    }
}

/// Validates a [`CanvasElement`]: its component reference and grid position.
pub fn validate_canvas_element(element: &CanvasElement, schema: &UiSchema) -> Result<(), Vec<ValidationError>> {
    let mut errors: Vec<ValidationError> = Vec::new();

    let component_exists = schema
        .components
        .iter()
        .chain(schema.shared_components.iter())
        .any(|c| c.id == element.component_id);

    if !component_exists {
        errors.push(ValidationError {
            path: format!("canvasElement.{}", element.id),
            message: format!("Referenced component '{}' not found", element.component_id),
            error_type: "reference".to_string(),
        });
    }

    if element.grid_position.column < 1 {
        errors.push(ValidationError {
            path: format!("canvasElement.{}.gridPosition.column", element.id),
            message: "Column must be >= 1".to_string(),
            error_type: "value".to_string(),
        });
    }

    if element.grid_position.row < 1 {
        errors.push(ValidationError {
            path: format!("canvasElement.{}.gridPosition.row", element.id),
            message: "Row must be >= 1".to_string(),
            error_type: "value".to_string(),
        });
    }

    if element.grid_position.col_span < 1 {
        errors.push(ValidationError {
            path: format!("canvasElement.{}.gridPosition.colSpan", element.id),
            message: "ColSpan must be >= 1".to_string(),
            error_type: "value".to_string(),
        });
    }

    if element.grid_position.row_span < 1 {
        errors.push(ValidationError {
            path: format!("canvasElement.{}.gridPosition.rowSpan", element.id),
            message: "RowSpan must be >= 1".to_string(),
            error_type: "value".to_string(),
        });
    }

    if errors.is_empty() {
        Ok(())
    } else {
        Err(errors)
    }
}

/// Validates a [`ComponentDef`]: required fields, event names, slot integrity.
pub fn validate_component(component: &ComponentDef) -> ValidationResult {
    let mut errors: Vec<ValidationError> = Vec::new();

    if component.id.is_empty() {
        errors.push(ValidationError {
            path: format!("components.{}", component.id),
            message: "Component ID cannot be empty".to_string(),
            error_type: "required".to_string(),
        });
    }

    if component.name.is_empty() {
        errors.push(ValidationError {
            path: format!("components.{}", component.id),
            message: "Component name cannot be empty".to_string(),
            error_type: "required".to_string(),
        });
    }

    for slot in &component.slots {
        if slot.is_empty() {
            errors.push(ValidationError {
                path: format!("components.{}.slots", component.id),
                message: "Slot name cannot be empty".to_string(),
                error_type: "required".to_string(),
            });
        }
    }

    for event_name in &component.events {
        if event_name.is_empty() {
            errors.push(ValidationError {
                path: format!("components.{}.events", component.id),
                message: "Event name cannot be empty".to_string(),
                error_type: "required".to_string(),
            });
        }
```

### 3f. `src/commands/sdui_commands.rs`

```rust
//! SDUI Tauri commands — invoked from tauri-front-shared (Angular).
//!
//! These commands expose the runtime engine to the Angular frontend via
//! Tauri's IPC bridge (`tauri::invoke`).

use std::sync::Arc;
use serde_json::Value;
use tauri::State;
use nosql_orm::provider::DatabaseProvider;

use crate::runtime::{
    SduiEngine, SchemaRouter, BindingResolver, validate_schema, validate_page,
    schema_types::{RenderedPage, ValidationResult},
};
use crate::schema::{UiSchema, Page};
use crate::storage::{SignalStore, JsonDb};
use crate::{AppError, Result};

/// Thread-safe shared state holding the active SDUI engine.
pub type SduiEngineState = Arc<std::sync::RwLock<Option<SduiEngine>>>;

/// Loads a [`UiSchema`] from the database by `schema_id` and initializes the
/// shared [`SduiEngine`] in the app's State.
///
/// Returns the schema metadata (id + version) on success.
#[tauri::command]
pub async fn sdui_load_schema(
    db: tauri::State<'_, impl DatabaseProvider>,
    engine_state: tauri::State<'_, SduiEngineState>,
    schema_id: String,
) -> Result<Value> {
    let data = db
        .find_by_id("schemas", &schema_id)
        .await
        .map_err(AppError::from)?
        .ok_or_else(|| AppError::NotFound(format!("Schema '{}' not found", schema_id)))?;

    let schema: UiSchema = serde_json::from_value(data)
        .map_err(|e| AppError::ValidationError(format!("Invalid schema format: {}", e)))?;

    let engine = SduiEngine::from(schema);

    let mut guard = engine_state.write().unwrap();
    *guard = Some(engine);

    Ok(serde_json::json!({
        "schemaId": schema_id,
        "schemaVersion": guard.as_ref().unwrap().schema.schema_version
    }))
}

/// Resolves a data binding expression against the current engine's store.
///
/// `binding_path` is the dot-notation path (e.g. `"user.address.city"`).
/// Returns the resolved JSON value, or `null` if any segment is missing.
#[tauri::command]
pub fn sdui_resolve_binding(
    engine_state: tauri::State<'_, SduiEngineState>,
    binding_path: String,
) -> Result<Value> {
    let guard = engine_state.read().unwrap();
    let engine = guard.as_ref().ok_or_else(|| AppError::Internal("Engine not initialized".to_string()))?;

    let resolver = BindingResolver::new(engine.store.clone(), Arc::new(engine.functions.clone()));
    resolver
        .resolve_path(&binding_path)
        .map(|opt| opt.unwrap_or(Value::Null))
}

/// Retrieves the rendered page for a given route `path`.
///
/// The router resolves the route config, evaluates guards, applies layout,
/// resolves bindings, and runs middleware — returning a [`RenderedPage`].
#[tauri::command]
pub fn sdui_get_page(
    engine_state: tauri::State<'_, SduiEngineState>,
    path: String,
) -> Result<RenderedPage> {
    let guard = engine_state.read().unwrap();
    let engine = guard.as_ref().ok_or_else(|| AppError::Internal("Engine not initialized".to_string()))?;

    let router = SchemaRouter::new(Arc::new(engine.clone()));
    router.resolve(&path)
}

/// Dispatches a component event by `event_name` on the element at `element_id`
/// on the page at `page_id`.
///
/// The event payload is passed to any registered event handlers in the engine's
/// function registry. Returns the updated element props after handler execution.
#[tauri::command]
pub fn sdui_dispatch_event(
    engine_state: tauri::State<'_, SduiEngineState>,
    page_id: String,
    element_id: String,
    event_name: String,
    payload: Value,
) -> Result<Value> {
    let guard = engine_state.read().unwrap();
    let engine = guard.as_ref().ok_or_else(|| AppError::Internal("Engine not initialized".to_string()))?;

    let handler_name = format!("on_{}_{}", element_id, event_name);
    let result = engine.call_function(&handler_name, vec![payload])?;

    let page = engine
        .get_page(&page_id)
        .ok_or_else(|| AppError::NotFound(format!("Page '{}'", page_id)))?;

    let element = page
        .canvas_elements
        .iter()
        .find(|e| e.id == element_id)
        .ok_or_else(|| AppError::NotFound(format!("Element '{}' on page '{}'", element_id, page_id)))?;

    Ok(element.props.clone().into_iter().collect::<Value>())
}

/// Validates the currently loaded schema in the engine.
///
/// Returns a [`ValidationResult`] with `valid: true` if no errors were found.
#[tauri::command]
pub fn sdui_validate_schema(
    engine_state: tauri::State<'_, SduiEngineState>,
) -> Result<ValidationResult> {
    let guard = engine_state.read().unwrap();
    let engine = guard.as_ref().ok_or_else(|| AppError::Internal("Engine not initialized".to_string()))?;

    Ok(validate_schema(&engine.schema))
}

/// Applies a theme by `theme_name` to the active engine.
///
/// Looks up the theme in `schema.app.settings.themes`. Returns the new theme
/// object on success, or an error if the theme is not found.
#[tauri::command]
pub fn sdui_apply_theme(
    engine_state: tauri::State<'_, SduiEngineState>,
    theme_name: String,
) -> Result<Value> {
    let mut guard = engine_state.read().unwrap();
    let engine = guard.as_mut().ok_or_else(|| AppError::Internal("Engine not initialized".to_string()))?;

    let new_theme = engine
        .schema
        .app
        .settings
        .themes
        .iter()
        .find(|t| t.name == theme_name)
        .ok_or_else(|| AppError::NotFound(format!("Theme '{}' not found", theme_name)))?
        .clone();

    engine.apply_theme(new_theme.clone());
    Ok(serde_json::to_value(&new_theme)
        .map_err(|e| AppError::Internal(format!("Failed to serialize theme: {}", e)))?)
}
```

## 8. Implementation Notes

### Thread Safety

`SduiEngine` is `Send + Sync` because all interior mutability (`SignalStore`, `HashMap<String, Callable>`) uses `Arc<RwLock>` or `Arc` wrappers internally. The `SduiEngineState` command parameter wraps `Arc<std::sync::RwLock<Option<SduiEngine>>>`, ensuring the engine is shared safely across async command invocations.

### SignalStore already exists

`SignalStore` (defined in `src/storage/signal_store.rs`) is already reactive with subscriber notification. The engine reuses it directly — no changes needed to storage.

### JsonDb persistence

`JsonDb` (`src/storage/json_db.rs`) can be used by the Angular side to persist the schema to a local JSON file on the device. The `sdui_load_schema` command reads from the nosql_orm database; an alternative command can read from `JsonDb` for offline-first scenarios.

### ts-rs integration

All new structs use `#[derive(TS)]` with `#[ts(export)]`. Running `cargo build` with the `ts-rs` macro expansion produces TypeScript bindings in `bindings/` (configured via `ts_rs::bindings` in the crate root or via a `build.rs` script if custom output path is needed).

### Guard/Middleware function registry

Guards and middleware are stored as strings (function names) in `RouteConfig` and resolved at runtime from `engine.functions`. This avoids serializing function references and allows runtime registration after schema load.

### No external runtime dependencies

The implementation relies only on `std`, `serde`, `serde_json`, `schemars`, `ts_rs`, and the existing `tauri-shared` modules. No additional crates are needed.
